Detailed Description

template<class eq>
struct inciter::deck::check_compns< eq >

Set defaults and do error checking on the compressible Navier-Stokes equation block.

This is error checking that only the compressible Navier-Stokes equation block must satisfy. Besides error checking we also set defaults here as this block is called when parsing of a compns...end block has just finished.
