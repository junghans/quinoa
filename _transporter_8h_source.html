18  asynchronous algorithm implemented in this class The detailed discussion of
19  the algorithm is given in the Charm++ interface file transporter.ci, which
20  also repeats the graph below using ASCII graphics. On the DAG orange
21  fills denote global synchronization points that contain or eventually lead
22  to global reductions. Dashed lines are potential shortcuts that allow
23  jumping over some of the task-graph under some circumstances or optional
24  code paths (taken, e.g., only in DEBUG mode). See the detailed discussion in
25  transporter.ci.
