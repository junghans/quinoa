private

Extract node IDs from element side sets and match to BCs.

Returns
List of owned node IDs on which a Dirichlet BC is set by the user

Boundary conditions, mathematically speaking, are applied on finite surfaces. These finite surfaces are given by element sets. This function takes the node lists mapped to side set IDs and extracts only those nodes that we own that the user has specified boundary condition on and returns the 'new' (as in producing contiguous-row-id linear system contributions, see also Partitioner.h) node IDs.
